# LibTabularasa

Small C++ library which allows to generate printable tables like this:

```
+-------+---------+---+
|  Foo  |     Bar | - |
+-------+---------+---+
| 12345 |       X | y |
|  123  | 12345-r | n |
| 2345  |       Y | n |
+-------+---------+---+
```

See unittest/TableTest.cpp for usage reference.
